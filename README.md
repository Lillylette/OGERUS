# Проект «ОГЭRUS — подготовка к ОГЭ по русскому языку»

### Описание проекта

**ОГЭRUS** — это интерактивное веб-приложение для подготовки к ОГЭ по русскому языку, разработанное на Python с использованием Flask. Приложение предоставляет пользователям площадку для изучения правил русского языка, тестирования знаний через систему интерактивных заданий, отслеживания прогресса и визуализации результатов обучения.

**Основные возможности:**
- Интерактивные задания с тремя уровнями сложности (без подсказок, с подсказками, полный разбор)
- Система предупреждений для сложных тем (н/нн в прилагательных, не с разными частями речи)
- Круговая диаграмма прогресса освоения навыков (орфография, пунктуация, анализ текста, изложение, сочинение)
- Чек-лист тем с сохранением прогресса
- Система достижений в профиле пользователя
- Отображение проблемных тем, где пользователь часто ошибается
- Избранное для сохранения заданий и памяток
- Скачивание памяток в формате PDF

**Возможности в самих заданиях и тестах:**
- Кликабельная пунктуация — расстановка знаков препинания кликами
- Коллекция карточек с правилами-исключениями
- Тесты с отложенным результатом (ответы видны только в конце)
- Режим проверяющего — поиск ошибок в готовых решениях
- Цветной режим для подсветки частей предложения

### Детальное описание файлов

#### Папка `app/` — исходный код приложения

**Основные файлы:**
- `__init__.py` — создание и настройка приложения Flask, инициализация расширений (SQLAlchemy, LoginManager, Migrate)
- `models.py` — модели  SQLAlchemy: User, Topic, Task, UserProgress, Favorite, Memo, ExceptionCard, UserCardCollection, Test, TestResult, Achievement, UserAchievement

**Модуль `auth/` — аутентификация:**
- `routes.py` — маршруты для регистрации, входа и выхода из системы

**Модуль `tasks/` — задания:**
- `routes.py` — маршруты для отображения заданий, отправки ответов, получения подсказок и разбора
- `logic.py` — функции проверки правильности ответов для разных типов заданий (кликабельная пунктуация, режим проверяющего, выбор ответа, ввод текста)

**Модуль `progress/` — прогресс:**
- `routes.py` — маршруты для получения статистики, данных для круговой диаграммы, обновления чек-листа
- `stats.py` — вычисление процента освоения навыков, определение проблемных тем (те, в которых ошибок >40%)

**Модуль `favorites/` — избранное:**
- `routes.py` — маршруты для добавления/удаления заданий и памяток в избранное, получения списка избранного

**Модуль `collections/` — коллекции:**
- `routes.py` — маршруты для получения карточек исключений, добавления в коллекцию после прохождения темы, отображения галереи коллекции

**Модуль `tests/` — тесты:**
- `routes.py` — маршруты для отображения тестов, сохранения ответов, подсчёта результатов и формирования разбора ошибок

**Папка `templates/` — HTML-шаблоны:**
- `base.html` — базовый шаблон с главным меню и подключением CSS/JS
- `index.html` — главная страница
- `profile.html` — профиль пользователя (вкладки: прогресс, достижения, проблемные темы, избранное, коллекция)
- `task.html` — страница задания с выбором уровня сложности, полем для ответа и кнопками
- `test.html` — страница теста с последовательными вопросами
- `memo.html` — страница памятки с кнопкой скачивания PDF

**Папка `static/` — статические файлы:**
- `css/style.css` — стили для всех страниц
- `js/punctuation.js` — логика кликабельной пунктуации (обработка кликов, выбор знаков, подсветка результатов)
- `js/progress_chart.js` — построение круговой диаграммы на Chart.js
- `js/checklist.js` — управление чек-листом тем, автосохранение прогресса
- `js/color_mode.js` — включение/выключение цветного режима, подсветка частей предложения

#### Папка `data/` — файлы данных

**JSON-файлы с контентом:**
- `tasks.json` — массив заданий с полями: id, topic_id, type, question, correct_answer, tips, full_explanation
- `topics.json` — темы с полями: id, name, skill, is_difficult, order
- `memos.json` — памятки с полями: id, topic_id, title, content
- `exception_cards.json` — карточки исключений с полями: id, title, rule_text, example, image_url, topic_id
- `tests.json` — тесты с полями: id, name, description, questions (массив вопросов с вариантами ответов), max_score
- `achievements.json` — достижения с полями: id, name, description, icon, condition_type, condition_value

**Файлы базы данных:**
- `initial_data.sql` — SQL-скрипт для начального заполнения таблиц (темы, достижения, тестовый пользователь)

#### Файлы конфигурации:
- `config.py` — настройки приложения: SECRET_KEY, SQLALCHEMY_DATABASE_URI, параметры сессий
- `requirements.txt` — список зависимостей: Flask, Flask-SQLAlchemy, Flask-Login, Flask-Migrate, WeasyPrint, bcrypt
- `run.py` — точка входа, запуск приложения Flask
